## Requirement
* PHP8.1.12
* Laravel9
* MySQL5.6
* Node 14.21.0
* npm6.14.17
* composer2.4.4

## Getting started
### clone
```
ex) $ git clone https://github.com/vfa-chau/playing-card.git
```

### Setup Laravel
```
$ cp .env.example .env
```

### Run docker containers
```
$ make start
```

### Setup node_modules and composer
```
$ make setup
```

### Run unit test
```
$ make test
```

### Run unit test coverage
```
$ make test_coverage
```

## Run local
- Local Server: localhost:8000
